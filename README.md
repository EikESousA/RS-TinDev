## 🎯 Descrição da Aplicação

<p> Aplicativo muito parecido com o tinder para desenvolvedores. </p> 

<p> O aplicativo foi organizado onde os usuários podem cadastrar e pesquisar outros desenvolvedores tanto pela web quanto mobile. </p> 

<p> A aplicação foi criada através do curso realizado pela 
  <img alt="logo rocketseat" src="./img/rocketseat.png" width="auto" heigth="auto"/>
  <a href="https://rocketseat.com.br/"> Rocketseat </a>
  (Semana Omnistack - 08), onde esse curso foi realizado no dia <strong> 05/08/2019 </strong> até o dia <strong> 09/08/2019 </strong>. 
</p>

<p>
  <strong> Back-end </strong>: Node.JS

## ⚡ Executando o Projeto

#### Clonando o projeto

```sh
$ git clone https://github.com/EikESousA/RS-TinDev.git
$ cd RS-BeTheHero
```

#### Iniciando a API

```sh
$ cd server
$ yarn && yarn dev
```

#### Iniciando o Front-end Web
```sh
$ cd web
$ yarn && yarn start
```

#### Iniciando o Front-end Mobile (Android)

```sh
$ cd mobile
$ yarn && yarn android && yarn start
```
